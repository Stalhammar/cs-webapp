Try Binary!

Computers think in 0s and 1s.

Each digit in a binary number represents a power of 2, starting from the right.
Multiply each digit by 2 raised to the power of its position, starting at 0, then add them all together.

For example, 1011 is:
(1 × 23) + (0 × 22) + (1 × 21) + (1 × 2 0) = 8 + 0 + 2 + 1 = 11

Guess the Bit Width!

Computers store numbers in binary — a system made up of only 1s and 0s. Each individual 1 or 0 is called a bit. The more bits you have, the larger the number you can represent.

For example, the number 5 in binary is 101, which takes 3 bits to store.
