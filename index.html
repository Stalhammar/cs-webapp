Try Binary!

Computers think in 0s and 1s.

Each digit in a binary number represents a power of 2, starting from the right.
Multiply each digit by 2 raised to the power of its position, starting at 0, then add them all together.

For example, 1011 is:
(1 × 23) + (0 × 22) + (1 × 21) + (1 × 2 0) = 8 + 0 + 2 + 1 = 11
